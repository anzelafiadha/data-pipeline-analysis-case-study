# Data pipeline analysis (case study)
This project presents a simplified case study on how raw data can be processed and transformed into meaningful insights.

A synthetic dataset representing schools, students, and teachers was created to simulate a real-world scenario. To reflect common data challenges, several issues were intentionally introduced, such as duplicate records, missing values, and inconsistent data entries.

The project then follows a structured data processing approach to clean and organize the dataset before performing analytical queries. These analyses aim to uncover patterns such as student distribution across schools, as well as highlight data quality issues that may impact decision-making.

Overall, this project demonstrates the importance of data preparation and structured analysis in turning imperfect data into useful information.
